# Etymology Explorer

**Etymology Explorer** is a web application that traces the evolution of English words through time and geography.  
Search any word to explore its linguistic ancestry—from Modern English back through historical languages to ancient roots.

## Features

- Real-time etymology lookup via the Wiktionary API  
- Visual language chain (Modern → Historical → Proto languages)  
- Interactive world map showing where each language was historically spoken  
- Chronological progression of word origins  
- Support for 25+ languages, including Proto-Indo-European, Latin, Old English, Sanskrit, and more

## WebSite

**[Etymology Explorer](https://etymology-explorer-app.vercel.app)**

## How It Works

1. Enter an English word  
2. Fetches etymological data from Wiktionary  
3. Parses the language lineage and approximate time periods  
4. Builds a chronological tree of linguistic descent  
5. Click any language node to view its geographic origin on the map

## Tech Stack

- **React 18**  
- **Wiktionary REST API**  
- **GeoJSON** for geographic visualization
